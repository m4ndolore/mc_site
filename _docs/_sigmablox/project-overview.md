# SigmaBlox Defense Tech Combine - Project Overview

## Mission

SigmaBlox is a digital platform for the first-ever technology combine for national security problem solvers. We connect defense tech companies with experienced coaches and evaluators to accelerate the path from innovation to impact. No fluff, no tourists - just warfighter problems, tactical solutions, and the hard path to deployment.

## Vision

To create the premier proving ground where defense technology companies demonstrate their capabilities, receive expert mentorship, and connect with the resources needed to scale their solutions for national security impact.

## Core Platform Components

### 1. Company Profiles & Discovery

- **Company Browser**: Filter and explore companies by mission area (Fires, Plans, C2, Autonomy)
- **Company Profiles**: Detailed profiles showing TRL levels, use cases, and problem statements
- **Profile Management**: Self-service updates through dedicated forms at /company-profile
- **Airtable Integration**: Real-time sync with company database

### 2. Coach Network

- **Coach Profiles**: Comprehensive profiles with SME areas, bootcamp roles, and contact info
- **Email-based Lookup**: Quick access to coach information via email search
- **Availability Tracking**: Track who's attending events and their travel details
- **Emergency Contacts**: Secure storage of critical contact information

### 3. Mission Area Categories

- **Fires**: Kinetic and non-kinetic effects delivery systems
- **Plans**: Mission planning and intelligence fusion tools
- **C2**: Battle management and communications infrastructure
- **Autonomy**: Unmanned systems and AI/ML applications

### 4. Platform Features

- **Embeddable Components**: Standalone company lists for external websites
- **Responsive Design**: Optimized for desktop and mobile viewing
- **Real-time Data**: Live updates from Airtable backend
- **Advanced Filtering**: Browse companies by category, TRL level, and use case

## Key User Types

### Coaches

- Experienced defense tech professionals
- Former military and government officials
- Industry experts with security clearances
- Technical advisors with domain expertise

### Entrepreneurs

- Defense tech startups
- Established companies pivoting to defense
- Research teams from universities
- Small businesses with dual-use technologies

### Program Administrators

- Incubator staff managing cohorts
- Security compliance officers
- Partnership coordinators

## Security Requirements

- Multi-level security clearance support
- ITAR compliance for technical data
- Encrypted data storage and transmission
- Audit trails for all information access

## Success Metrics

- Time to contract award for participating companies
- Knowledge transfer effectiveness
- Coach-entrepreneur match satisfaction
- Security incident prevention rate
- Platform engagement and utilization

## Technical Architecture Priorities

- Scalable microservices architecture
- Role-based access control (RBAC)
- Real-time collaboration capabilities
- Mobile-responsive design
- Integration with government systems (SAM.gov, CAGE codes, etc.)
