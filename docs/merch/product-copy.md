## Brand Voice

All product copy must adhere to **operational tone**:

| DO | DON'T |
|----|-------|
| Directive statements | Marketing fluff |
| Tactical language | Consumer hype |
| Minimal qualifiers | Charity framing |
| Action-oriented | Passive voice |
| Mission-focused | Lifestyle branding |

## Copy Anti-Patterns

**AVOID these phrases:**

| Anti-Pattern | Why |
|--------------|-----|
| "Support our troops" | Charity framing |
| "Give back" | Charity framing |
| "Make a difference" | Consumer marketing |
| "Join the movement" | Consumer marketing |
| "Limited edition" | Artificial scarcity hype |
| "Premium quality" | Empty marketing speak |
| "Exclusive access" | Consumer FOMO tactics |
| "Shop now" | E-commerce CTA speak |

**USE these instead:**

| Alternative | Why |
|-------------|-----|
| "Build" | Action-oriented |
| "Deploy" | Tactical language |
| "Signal" | Communication theme |
| "Directive" | Mission-focused |
| "Standard-issue" | Military/ops context |
| "Field-ready" | Operational readiness |

## SEO Metadata

### Title Format
`{Product Name} | Merge Combinator`

### Meta Description Format
`{Short Description} — {Material/Category} from Merge Combinator.`

Example:
- Title: "Build What Warfighters Need Tee | Merge Combinator"
- Meta: "The directive. No qualifiers. — T-shirt from Merge Combinator."
