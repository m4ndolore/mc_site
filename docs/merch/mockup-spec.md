# Merge Combinator Merch Mockup Spec (v1)

## Brand Placement Rules (Global)
- Primary chest mark: **Full Merge Combinator logo (text + arrows)**.
- Print assets: `docs/merch/assets/mc-logo-full.png` (hi-res), `docs/merch/assets/mc-logo-full.svg` (editable text + arrows image).
- Arrows-only mark: `docs/merch/assets/mc-arrows.png` and `docs/merch/assets/mc-arrows.svg`.
- Secondary back mark: **Arrows only**, small, placed **just below the back collar**.
- No gradients, no drop shadows, no outlines.
- Use single‑color prints (white on dark, black on light).

## Typography Rules
- If using text‑only designs (e.g., BWWN, RAD), use **Inter** or **Space Grotesk** in all caps.
- Tracking: +40 to +80 (wide).
- Weight: 600–700.

## Placement Specs (Print‑Ready)
### Chest (Full Logo)
- Position: Left chest.
- Width: 3.0–3.5 in (76–89 mm).
- Height: proportionate (do not stretch).
- Top edge: 2.5–3.0 in (64–76 mm) below collar seam.
- Centered horizontally on left chest panel.

### Back (Arrows Only)
- Position: Centered, just below collar seam.
- Width: 1.5–1.75 in (38–45 mm).
- Top edge: 1.0–1.25 in (25–32 mm) below collar seam.

## Product‑Specific Notes
### Polos
- Chest: Full logo, 3.0 in wide.
- Back: Arrows only, 1.5 in wide.
- Thread/ink: tone‑matched white or light gray on dark fabrics.

### Hawaiian Shirts
- Keep logos minimal; chest print only.
- Back arrows optional, only if pattern density is low.

### T‑Shirts
- Chest: Full logo, 3.25 in wide.
- Back: Arrows only, 1.75 in wide.

### Patches
- Primary patch: Arrows only or “BWWN/RAD” text.
- Size: 3.0 in wide (standard).

### Sticker Patches
- Arrows only, 2.0 in wide.
- Matte finish preferred.

## Colorways
- Black
- Charcoal
- Slate
- Navy
- Sand
- Olive (optional for caps)
