# Sticker Print Placement Sheet

**Product Category:** Sticker Patches (MC-STICKER-*)
**Document Version:** 1.0
**Date:** 2026-01-30

---

## Overview

Stickers feature simple, single-design marks:
- **Arrows-only mark**
- **Arrows + bar** (Ops Console variant)
- **Text designs** (BWWN)

All stickers are die-cut to shape.

---

## Standard Sticker Layout

```
    ┌─────────────────────────────────┐
    │                                 │
    │     ┌─────────────────────┐     │
    │     │                     │     │
    │     │    PRIMARY DESIGN   │     │    ← Die-cut to shape
    │     │    (arrows / text)  │     │
    │     │                     │     │
    │     └─────────────────────┘     │
    │                                 │
    └─────────────────────────────────┘
            │←── 2.0 in ──→│
```

### Dimensions

| Specification | Value |
|---------------|-------|
| **Sticker Width** | 2.0 in (51 mm) |
| **Sticker Height** | Proportionate to design |
| **Bleed** | 0.0625 in (1.5 mm) |
| **Die Line** | Follows design contour |

## Production Notes

1. **Single-color designs** - Simple, bold marks
2. **Matte finish preferred** - Reduces glare, tactical aesthetic
3. **Outdoor-rated** - Must survive vehicle/equipment use
4. **Die line** - 0.0625 in offset from design edge
5. **No gradients** - Solid colors only
