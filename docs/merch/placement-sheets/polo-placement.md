# Polo Print Placement Sheet

**Product Category:** Polos (MC-POLO-*)
**Document Version:** 1.0
**Date:** 2026-01-30

---

## Overview

All polo shirts receive:
- **Front:** Full Merge Combinator logo (text + arrows) on left chest
- **Back:** Arrows-only mark below collar

---

## Front Placement (Left Chest)

```
         ┌─────────────────────────┐
         │          COLLAR         │
         └─────────────────────────┘
              │           │
              │   FRONT   │
         ┌────┴───────────┴────┐
         │                     │
         │  ┌─────────────┐    │
         │  │  FULL LOGO  │    │    ← 2.5-3.0 in below collar seam
         │  │  (text +    │    │
         │  │   arrows)   │    │
         │  └─────────────┘    │
         │                     │
         │                     │
         └─────────────────────┘
```

### Dimensions

| Specification | Value |
|---------------|-------|
| **Logo Width** | 3.0 in (76 mm) |
| **Logo Height** | Proportionate (do not stretch) |
| **Top Edge from Collar Seam** | 2.5 - 3.0 in (64 - 76 mm) |
| **Horizontal Position** | Centered on left chest panel |

### Print Method

| Fabric Color | Print Color |
|--------------|-------------|
| Black | White |
| Slate | White |
| Navy | White |
| Charcoal | White / Light Gray |
| Sand | Black |

## Production Notes

1. **Single-color prints only** - No gradients, no drop shadows, no outlines
2. **Thread/ink:** Tone-matched white or light gray on dark fabrics
3. **Outline fonts before final export** if using SVG
4. **PNG logo is large enough** for 3" wide prints at 300 DPI
