# Hawaiian Shirt Print Placement Sheet

**Product Category:** Hawaiian Shirts / Aloha (MC-ALOHA-*)
**Document Version:** 1.0
**Date:** 2026-01-30

---

## Overview

Hawaiian shirts receive minimal branding:
- **Front:** Full Merge Combinator logo (text + arrows) on left chest
- **Back:** Arrows-only mark below collar (OPTIONAL - only if pattern density is low)

---

         ┌─────────────────────────┐
         │          COLLAR         │

         ┌────┴───────────┴────┐

### Dimensions

| Specification | Value |
|---------------|-------|
| **Logo Width** | 2.5 - 3.0 in (64 - 76 mm) |
| **Logo Height** | Proportionate (do not stretch) |
| **Top Edge from Collar Seam** | 2.5 - 3.0 in (64 - 76 mm) |
| **Horizontal Position** | Centered on left chest panel |

### Design Guidance

- Keep logos minimal to avoid visual competition with pattern
- Logo should read clearly against patterned background
- Consider pattern density when selecting logo size

### Dimensions (if used)

| Specification | Value |
|---------------|-------|
| **Arrows Width** | 1.25 - 1.5 in (32 - 38 mm) |
| **Arrows Height** | Proportionate |
| **Top Edge from Collar Seam** | 1.0 - 1.25 in (25 - 32 mm) |

### When to Include Back Arrows

| Pattern Type | Back Arrows |
|--------------|-------------|
| Dense/busy pattern | **Skip** - logo won't read |
| Medium density | Evaluate individually |
| Minimal/sparse pattern | Include |
| Solid color panels | Include |

## Production Notes

1. **Single-color prints only** - No gradients, no drop shadows, no outlines
2. **Chest print only** is acceptable - back arrows are optional
3. **Pattern evaluation required** before finalizing back placement
4. **Outline fonts before final export** if using SVG
5. **Match print color to pattern tones** for cohesion

---

## Pattern Guidelines

- Pattern should be "quiet" / "minimal" per brand guidelines
- Avoid tropical cliches - aim for tactical/operational aesthetic
- Patterns should complement, not compete with, MC branding
