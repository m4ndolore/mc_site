# Operator Notebook (MC-NOTE-OPS)

## Overview

Grid-ruled notebook with minimal branding:
- **Front Cover:** Full Merge Combinator logo
- **Back Cover:** None or minimal arrows mark
- **Interior:** Grid ruling (5mm or dot grid)

## Production Notes

### Notebook

1. **Matte black cover** - No gloss
2. **Foil stamp preferred** - White or silver foil on black
3. **Grid interior** - Engineer/ops aesthetic
4. **Quality paper** - No bleed-through

### Desk Mat

1. **Subtle design** - Grid + watermark, not busy
2. **Black/gray palette** - Operational, not decorative
3. **Stitched edge** - Durability
4. **Non-slip base** - Essential for ops use
