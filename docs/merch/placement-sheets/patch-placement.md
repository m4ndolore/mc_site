# Patch Print Placement Sheet

**Product Category:** Patches (MC-PATCH-*)
**Document Version:** 1.0
**Date:** 2026-01-30

---

## Overview

Patches feature either:
- **Arrows-only mark**, or
- **Text designs** (BWWN, RAD)

All patches are single-sided, woven/embroidered.

---

## Standard Patch Layout

```
    ┌─────────────────────────────────┐
    │                                 │
    │                                 │
    │     ┌─────────────────────┐     │
    │     │                     │     │
    │     │    PRIMARY DESIGN   │     │    ← Centered
    │     │    (arrows / text)  │     │
    │     │                     │     │
    │     └─────────────────────┘     │
    │                                 │
    │                                 │
    └─────────────────────────────────┘
            │←─── 3.0 in ───→│
```

### Dimensions

| Specification | Value |
|---------------|-------|
| **Patch Width** | 3.0 in (76 mm) |
| **Patch Height** | Proportionate to design (typically 1.5 - 2.0 in) |
| **Border Width** | 0.125 in (3 mm) merrowed edge |
| **Design Margin** | 0.25 in (6 mm) from edge |

## Design Variants

### MC-PATCH-BWWN (Build What Warfighters Need)

```

    │                                 │
    │   BUILD WHAT WARFIGHTERS NEED   │

    └─────────────────────────────────┘
```

| Specification | Value |

```
    │           ▲  ▲                  │    ← Arrows mark
```

## Production Method

| Method | Notes |
|--------|-------|
| **Woven** | Preferred for fine detail |
| **Embroidered** | Acceptable for text-heavy designs |
| **PVC/Rubber** | Alternative for tactical look |

### Attachment Options

- Hook-and-loop (Velcro) backing - default
- Iron-on backing - optional
- Sew-on - optional

## Production Notes

1. **Single-color designs only** - Black background with light design
2. **Merrowed edge** - Heat-cut and locked edge
3. **Velcro backing** - Hook side attached to patch
4. **Typography** - Outline fonts before production
5. **No gradients** - All solid colors
