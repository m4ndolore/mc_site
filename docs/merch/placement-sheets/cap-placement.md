# Cap Print Placement Sheet

**Product Category:** Caps (MC-CAP-*)
**Document Version:** 1.0
**Date:** 2026-01-30

---

## Overview

Field caps receive:
- **Front:** Full Merge Combinator logo (text + arrows) on left front panel
- **Back:** None (standard adjustable strap area)

Low-profile, signal-only design.

---

## Print/Embroidery Method

| Method | Notes |
|--------|-------|
| **Embroidery** | Preferred for durability |
| **Heat transfer** | Alternative for fine detail |
| **Screen print** | Not recommended (cap material) |

### Embroidery Specifications

| Property | Value |
|----------|-------|
| **Stitch count** | ~8,000 - 12,000 |
| **Thread** | Polyester |
| **Backing** | Tear-away or cut-away |
| **Density** | Medium (avoid puckering) |

## Production Notes

1. **Low profile** - No tall/trucker style
2. **Embroidery preferred** - More durable than print
3. **Single-color logo** - Match to cap tone
4. **Outline fonts** - Convert text to paths before digitizing
5. **Test stitch** - Verify density doesn't pucker fabric
