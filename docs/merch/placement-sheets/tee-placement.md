# T-Shirt Print Placement Sheet

**Product Category:** T-Shirts (MC-TEE-*)
**Document Version:** 1.0
**Date:** 2026-01-30

---

## Overview

All t-shirts receive:
- **Front:** Full Merge Combinator logo (text + arrows) on left chest
- **Back:** Arrows-only mark below collar

---

## Production Notes

1. **Single-color prints only** - No gradients, no drop shadows, no outlines
2. **Slightly larger chest logo** than polos (3.25 in vs 3.0 in)
3. **Slightly larger back arrows** than polos (1.75 in vs 1.5 in)
4. **Outline fonts before final export** if using SVG
5. **PNG logo is large enough** for 3.25" wide prints at 300 DPI
